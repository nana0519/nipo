# nipo

## サイト概要
建設会社向けの作業日報と勤務表の作成、管理ができるWEBアプリケーション

## ターゲットユーザ
- 日報をペーパーレス化したい建設会社
- 業務を効率化させたい建設会社

## 目指した課題解決
業務の効率化を目的とした日報のペーパーレス化と管理
